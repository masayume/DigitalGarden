## compile
```
rustc 04-print.rs -o 04-print.exe
```

## create cargo package
```
mkdir rust-package
cd rust-package
cargo init

```

## compile cargo package
```
cd package-dir
cargo build
cargo build --release
cargo run
```

## call with arguments
```
cargo run param1 param2
```
